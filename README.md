 Network Scanner

A Python-based network scanning tool that discovers active hosts,
open ports, and running services on a target network.
Built as a cybersecurity portfolio project.



 Overview

Network scanning is a fundamental skill in cybersecurity — used by
security professionals to audit networks, find vulnerabilities, and
map infrastructure before attackers do. This tool automates that process.



 Features

-  Scan target IP or IP range for active hosts
-  Detect open ports on discovered hosts
-  Identify running services on open ports
-  Save scan results to results.txt
-  Fast and lightweight — pure Python



 Tech Stack

| Layer | Technology |
|---|---|
| Language | Python 3.x |
| Networking | socket, ipaddress |
| Scanning | Custom scanner module |
| Output | Text file (results.txt) |
